Сайт Салям инфо на GitHub

Удалось после многочисленных попыток зарегистрироваться на GitHub. Проблема заключалась в том что ресурс на котором расположена капча блокируется провайдерами интернета, хотя сам GitHub работает. Пришлось обращаться к специалисту для решения этой проблемы возникающей при регистрации.

salaminfo.github.io

В HTML5 писать закрывающий тег параграфа или абзаца необязательно, если следом идёт новый тег параграфа или абзаца. Но для правильного отображения во всех браузерах особенно старых рекомендуют писать закрывающие теги.
